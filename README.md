# Sistema de Vendas

Sistema desktop para gerenciamento de clientes, produtos, estoque e vendas, desenvolvido em Python utilizando PySide6 e SQLite.

## Funcionalidades

- Cadastro de clientes
- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Geração de pedidos
- Sistema de login
- Banco de dados SQLite
- Interface gráfica desenvolvida com PySide6

## Tecnologias Utilizadas

- Python
- PySide6
- SQLite
- Git
- GitHub

Este projeto está sendo desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas e da minha transição de carreira para a área de Desenvolvimento de Software.
